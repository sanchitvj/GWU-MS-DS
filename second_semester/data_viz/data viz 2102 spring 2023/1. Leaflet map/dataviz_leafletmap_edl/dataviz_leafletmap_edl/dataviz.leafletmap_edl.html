Leaflet map

I grew up in Sioux City, Iowa and attended Sioux City East High School. Here’s a map of the school.

# load leaflet library
library(leaflet)

# create map object
m <- leaflet() %>%
  addTiles() %>%  # Add default OpenStreetMap map tiles
  addMarkers(-96.3403, 42.4524, popup="Home of the Black Raiders")
m  # Print the map

I found the latitude and longitude coordinates by googling “Sioux City East latitude longitude.” You can do the same to find your own coordinates.

If you tried to knit this file and it gave you an error of Error in library(tm) : there is no package called ‘tm’ then you need to install the leaflet package. You can do that by clicking on the package tab in the lower right window, then the install button, and then type in leaflet. Hit return, and the package will install.
